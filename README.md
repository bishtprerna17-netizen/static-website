
<# Static Website on AWS EC2

This project is a simple static website hosted on an AWS EC2 instance using Nginx web server.

## Tech Stack
- HTML
- CSS
- AWS EC2
- VPC Networking
- Nginx
- Git & GitHub

## Deployment
The website is deployed on an EC2 instance and served using Nginx.

## What I Learned
- How to launch EC2 instance
- How to configure VPC networking
- How to host website using Nginx
- Basic Git & GitHub usage
